# AI-Driven Disease Outbreak Prediction System for Ghana

## Project Overview

This project implements an AI-driven disease outbreak prediction system specifically designed for Ghana, utilizing machine learning models to forecast disease outbreaks based on historical health data, meteorological conditions, and socio-economic factors.

## Objectives

- Develop predictive models for disease outbreak forecasting
- Create a web-based interface for real-time predictions
- Implement data ingestion capabilities for both synthetic and real-world data
- Provide actionable insights for public health decision-making

## Methodology

### Data Pipeline
- Data collection from multiple sources (health records, weather data, population statistics)
- Data preprocessing including feature engineering and encoding
- Synthetic data generation for model training

### Machine Learning Models
- Random Forest Classifier for baseline predictions
- XGBoost Classifier for improved accuracy
- Model evaluation using stratified cross-validation
- Feature importance analysis

### System Architecture
- Flask REST API for backend services
- Streamlit dashboard for user interface
- SQLAlchemy for data persistence
- Modular design for scalability

## Results

The system achieves:
- High accuracy in outbreak prediction
- Real-time prediction capabilities
- User-friendly web interface
- Extensible architecture for additional features

## Technologies Used

- Python 3.14
- Scikit-learn, XGBoost
- Flask, Streamlit
- Pandas, NumPy
- SQLAlchemy
